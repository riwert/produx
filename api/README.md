# scandiweb/swapi

**SWAPI** - a simple REST API in PHP (OOP)

## Endpoints:
- ```/product/list``` (without any param)
- ```/product/get``` (with sku url param)
- ```/product/massDelete``` (with comma separated skus url or body param)
- ```/product/delete``` (with sku url or body param)
